# AlbumMaker
Slack に写真を上げると、そっから Google Photo に上がって、その生の画像の URL を Dropbox Paper の対象のドキュメントに追記してくれる。
jpeg、png、gif 等ならいけるっぽい。

## 動作要件
中から sh と curl を呼び出してるので、この2つが無いと動かない

## 使い方
1. 環境変数に色々指定して起動する、すると…
2. 動く
